Intégrale de chemin

Mouvement Brownien selon les physiciens

On définit l'action suivante sur l'ensemble des chemins sur [0,1] :

$$S_{BM}(\sigma):=\frac{1}{2}\int_{0}^{1}\left|\sigma'(r)\right|^2dr$$

Puis on définit le fonctionnel du mouvement Brownien :

$$\mathbb{E}[F((B_s)_{0\leq s\leq 1})]:=\frac{1}{Z}\int_{\Sigma}F(\sigma)e^{-S_{BM}(\sigma)}D\sigma$$

Remarque : minimiseur de l'action = solution classique.

Intégrale de surface (cas de la sphère)

On définit l'action de Liouville :

$$S(X,\hat{g}):=\frac{1}{4\pi}\int_{\Sigma}(|\partial^{\hat{g}}X|^2+QR_{\hat{g}}X+4\pi\mu e^{\gamma X})\lambda_{\hat{g}}$$

Puis on définit le fonctionnel suivant :

$$\mathbb{E}[F(X)]:=\frac{1}{Z}\int F(X)e^{-S(X,g)}DX$$

On obtient le champ de Liouville.

Formellement, la mesure de Liouville s'écrit [à quelques détails près...]

$$e^{\gamma X}dg, \gamma\in]0,2[\cup\{2\}$$

Problème : donner un sens mathématique à cette définition !

Intégrale de surface (cas du disque)

On définit l'action de Liouville :

$$S(X,\hat{g}):=\frac{1}{4\pi}\int_{\Sigma}(|\partial^{\hat{g}}X|^2+QR_{\hat{g}}X+4\pi\mu e^{\gamma X})\lambda_{\hat{g}}+\frac{1}{2\pi}\int_{\partial\Sigma}(QK_{\hat{g}}X+2\pi\mu_\partial e^{\frac{\gamma}{2} X})\lambda_{\partial\hat{g}}$$

Puis on définit le fonctionnel suivant :

$$\mathbb{E}[F(X)]:=\frac{1}{Z}\int F(X)e^{-S(X,g)}DX$$

Difficulté : insertions au bord etc.

Processus Gaussien

Le mouvement Brownien sur $\mathbb{R}_+$ est un processus Gaussien centré de fonction de covariance $K(s,t)=s\wedge t$. [Exercice : calculer $\frac{d^2}{dx^2}K$.]

Fonction de Green

Soit $D\in\mathbb{C}$ un compact simplement connexe d'intérieur non vide. La fonction de Green (à une constante près) : $G_D(x,y)\approx -\log|x-y|$. Elle vérifie $\Delta G_D=\delta_{x,y}$.

Considérons un champ Gaussien X (un vecteur Gaussien indexé par D) de covariance G.

Exercice : calculer la variance $\mathbb{E}[X(x)^2]=G(x,x)$.

La régularité de X est mauvaise : X vit dans l'espace des distributions sur D.

Question : comment définir l'exponentiel de X ?

Idée #2 : régularisation et renormalisation


$X(x)$ n'est pas défini, mais on peut regarder la moyenne de $X$ sur $B(x,\epsilon)$ (ou $\partial B(x,\epsilon)$). De manière générale, on note $X_\epsilon$ le champ obtenu après cette convolution.
Pour chaque Borélien $A$, on définit $$M^{\epsilon}_g(A)=\int_{A}e^{\gamma X_\epsilon(x)-\frac{\gamma^2}{2}\mathbb{E}[X_\epsilon(x)^2]}dg.$$

Espérons que cette mesure aléatoire converge...

Chaos Multiplicatif Gaussien

Grâce à [Kahane 85'] :

Soit $X$ un champ Gaussien log-corrélé. Pour tout $\gamma\geq 0$, on peut définir $e^{\gamma X}$ comme une mesure aléatoire sur $D$. Cette mesure est non-dégénérée si et seulement si $\gamma<\sqrt{4}$.

En dimension 2, le GFF est un chaos multiplicatif !

Invariance conforme

Pour le mouvement brownien : théorème de Lévy.

Soit $B_t$ un mouvement Brownien dans $D\in\mathbb{C}$. Soit $\phi:D\to\mathbb{C}$ analytique, alors $\phi(B_t)$ est un mouvement Brownien changé de temps dans $\phi(D)$.

Invariance conforme pour le champ de Liouville

$(\psi,g)=(\psi\circ\phi+Q\ln\left|\phi'\right|,g\circ\phi)$ avec $Q=\frac{2}{\gamma}+\frac{\gamma}{2}$.
Remarque : théorie de Liouville classique : $Q=\frac{2}{\gamma}$.
